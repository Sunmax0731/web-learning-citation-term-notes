# Competitive Benchmark

## 比較対象

- Readwise
- Notion Web Clipper
- Hypothes.is
- YouTube notes extensions

## 採用する評価基準

- 入力から検証、証跡、再実行までが1つの流れであること。
- 手動確認が必要な項目を warning として残せること。
- 公開前に docs、release checklist、security/privacy を確認できること。
- ページ、動画、用語、引用を同じ学習ノートとして保存する。

## 差別化

この repo は Chrome拡張 Side Panel + local notes に閉じた最小実装だけでなく、QCDS、traceability、manual test、docs ZIP を含めた公開前の運用品質を製品の一部として扱う。
